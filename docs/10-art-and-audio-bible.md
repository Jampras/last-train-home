# 10 - Art and Audio Bible

## Visual Goal

Create a warm, painterly 2D world that feels alive, handmade, and emotionally safe by day, then uneasy and suspenseful by night.

The visual direction should be inspired by the softness and readability of handcrafted 2D animation, but it must not imitate the exact shapes, palettes, or character language of any direct reference.

## Visual Pillars

### 1. Silhouette First

Every important game element must read clearly in side view:

- leader cat
- recruit cats
- dogs
- human captors
- build nodes
- lamps
- train wagons

### 2. Warm Interior, Cold Exterior

The train must look warmer and more inviting than the world outside it.

- amber lights
- soft cloth
- rounded shapes
- wood and worn metal
- living details like mugs, blankets, plants, rope, and patched fabric

Stations should feel:

- exposed
- lonely
- weathered
- uncertain

### 3. Layered Atmosphere

The world should use:

- foreground silhouettes
- midground gameplay lanes
- parallax backgrounds
- weather overlays
- lamp glow and soft fog

### 4. Meaningful Motion

Animation should emphasize:

- tail movement
- ear reactions
- breathing and idle weight
- cloth flutter
- train vibration
- lamp flicker

Avoid noisy constant motion.

## Character Design Rules

### Cats

- readable class silhouettes
- distinct coats, clothing, bags, and tools
- avoid identical body shapes
- use class props to communicate role

Examples:

- engineer: gloves, tool belt, stronger silhouette
- lamplighter: lamp pole, long coat, tall profile
- tailor: satchel, thread kit, softer layered cloth
- scout: lighter frame, hood, field notes

### Humans

- no hyper-real detail
- readable shapes by role
- use lanterns, nets, whistles, masks, coats
- faces can stay partially obscured to maintain mystery

### Dogs

- must read as trained and threatening
- silhouettes must differ by role
- tracker, rush, guard, and alpha must all be identifiable at a glance

## Environment Art Rules

Station components:

- platforms
- lamps
- fencing
- signs
- rail debris
- signal posts
- crates
- shelters
- puddles

Biome shifts should be achieved through:

- palette
- weather
- background shapes
- lane obstacles
- signage and architecture

## Color Direction

Safe palette families:

- amber
- dusty cream
- slate blue
- petrol green
- rust red
- coal gray

Night contrast should come from:

- selective lamp warmth
- soft moonlight
- fog density
- silhouette edges

Avoid neon-heavy color language unless reserved for rare mystery events.

## UI Style

- low chrome
- slim frames
- hand-crafted texture touches
- clear typography
- no generic dashboard panels
- use chips, tags, stitched frames, signal-plate motifs

## Audio Direction

Audio should support suspense and warmth without becoming overwhelming.

Core layers:

- train ambience
- station ambience
- day music
- night tension music
- event cue stingers
- interaction foley

## Event Audio Rules

Each event class needs a unique recognizable signature:

- merchant: soft bell plus wagon rattle
- positive find: light chime
- mystery event: distant tonal ring or warped music-box color
- danger event: whistle, low drum, or hound cue

The player should start recognizing event types by ear.

## Music Goals

Day:

- reflective
- melodic
- lightly hopeful

Night:

- restrained tension
- pulse without panic
- space for event sounds

Train hub:

- intimate
- restorative
- familiar thematic motifs

## Asset Production Guidance

Recommended output categories:

- layered backgrounds
- station tile pieces
- wagon modules
- cat animations
- human animations
- dog animations
- event VFX
- UI icons
- dialogue portraits only if needed

## Animation Priority

Must-have animation set:

- player walk
- player idle
- interact
- leader skill
- recruit work loop
- human advance
- dog run
- lantern flicker
- train idle

Nice-to-have later:

- special emotional acting shots
- biome-specific idle variations
- extended merchant staging
