# 04 - Meta Progression and Content

## Meta Progression Philosophy

Permanent progression should grant:

- more options
- more identity
- more variety
- more visible train growth

Permanent progression should not trivialize the core run loop by stacking raw stats too early.

## The Train as Persistent Hub

The train is the main progression surface.

Between runs the player can:

- talk to rescued cats
- build or upgrade wagons
- choose the next route
- switch leaders
- review clues and memories
- equip train-wide support relics

The train must physically change over time.

Examples of visible train evolution:

- more wagons
- stronger lighting
- repaired roofs
- hanging cloth banners
- added storage
- plants, blankets, and signals
- cats occupying new workspaces

## Permanent Reward Types

- new leader cats
- wagon blueprints
- mastery upgrades
- relics added to future run pools
- route unlocks
- station biome unlocks
- story logs and memory scenes

## Leader Cat System

Each leader has:

- a personality
- a station playstyle
- a unique wagon or build line
- a mastery challenge
- an advanced upgrade unlocked by challenge completion

## Mastery Design Rules

- the cat is useful before mastery completion
- mastery upgrades deepen identity, not basic viability
- challenges are readable and trackable
- challenges encourage the intended playstyle
- challenge progress can be partial when appropriate

## Relic System

Relics are run modifiers unlocked permanently, then found situationally during runs.

Relic design rules:

- clear, short descriptions
- obvious synergy with a leader or structure type
- low count per run
- no overwhelming item spam

Example relics:

- `Warm Bell`: merchants stay longer
- `Soot Charm`: train repairs cost less scrap this run
- `Milk Token`: rescued kittens recover faster
- `Silent Thread`: first capture each night is prevented

## Unlock Economy

Recommended permanent currencies:

- `Memory Tokens` for story and leader unlocks
- `Blueprint Fragments` for wagons and structures
- `Route Marks` for world map progression

## Progression Promise Per Run

Every run should provide at least one of:

- a new unlock
- meaningful challenge progress
- visible train growth
- route advancement

If a run provides none of these, the game will feel stagnant.
